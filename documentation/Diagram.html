Component Deep Dive

Data Source & Local Dev

The process begins in a controlled environment. The Wokwi Simulator mimics a real-world IoT device, generating sensor data. To bridge the gap between the cloud-based simulator and a local machine, Ngrok creates a secure public URL that tunnels directly to the developer's machine.

Backend Processing

The Flask API, running locally in VS Code, acts as the system's central nervous system. It listens for incoming data from the Ngrok tunnel, validates it, and then securely transmits it to the cloud database for persistent storage.

Cloud & Frontend

Firebase Realtime Database serves as a flexible, scalable, and real-time data store. The React Dashboard subscribes to this database. This direct connection allows for instantaneous updates on the user's screen whenever new data is saved, creating a seamless and dynamic user experience.
